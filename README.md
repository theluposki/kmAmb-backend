# kmAmb-backend

### instalação

```bash
   $ git clone https://github.com/theluposki/kmAmb-backend.git

   $ cd kmAmb-backend/

   $ npm install

   $ npm start
```
### configurações

  é necessario configura o arquivo ```.env```

  ```bash 
    $ nano .env
  ```

  demonstração

  ![.env](readme-files/imgEnv.png)

  altere as variaveis de ambiente.

     ` DB_HOST=mongodb://localhost:27017/nomedobanco `
